# 🏟️ Stadium Manager — Java

> A stadium management system built with object-oriented Java.

A console-based Java application that applies OOP principles to a real-world stadium management scenario. Manage events, seating and operations through a structured class hierarchy with file-based data persistence.

## 📦 What's Inside

- 🏗️ Object-oriented class structure for stadium entities
- 📋 Event and seating management
- 💾 File system persistence for data storage
- 🖥️ Console-based interactive program flow
- 🧱 OOP principles — inheritance, encapsulation, polymorphism
